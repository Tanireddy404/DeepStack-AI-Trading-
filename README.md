# DeepStack-AI-Trading-
An AI agent that autonomously trades cryptocurrencies using Deep Reinforcement Learning to maximize portfolio returns.
DeepStack is a Deep Reinforcement Learning project that trains an AI agent to autonomously trade four major cryptocurrencies — Bitcoin, Ethereum, Cardano, and Solana — using a Deep Q-Network (DQN). The agent learns optimal buy, sell, and hold strategies by interacting with a simulated trading environment built on 2 years of real historical market data.
